# Transponder Squawk Codes

Reference list of standard aviation **transponder (SSR) squawk codes** — the four-digit codes pilots set on their transponder, including the emergency codes every pilot memorises: `7500` hijack, `7600` lost comms, `7700` emergency.

JSON and CSV, for flight-sim tools, ATC trainers and aviation apps.

## Files
- [`squawk-codes.json`](squawk-codes.json) · [`squawk-codes.csv`](squawk-codes.csv)

> Note: many squawk codes are assigned by ATC and vary by country/region. This list covers the internationally standardised and common conspicuity codes.
